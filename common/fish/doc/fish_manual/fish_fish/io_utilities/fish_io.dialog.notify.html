io.dialog.notify

Syntax

i = io.dialog.notify(i1,i2,s)

General notification event. The exact behavior of this function is dependent on the current user interface and is intended to serve as a general notification system to notify the user interface of kernel events. It is not currently used by the default Itasca interface.
