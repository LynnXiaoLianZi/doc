string.csv.from

Syntax

l := string.csv.from(s<,sep>)

Separate the string into a token list using the comma-separated-values convention.

By default, the separator used will be a comma character ,, otherwise the separator will be the string provided.

Returns:l - A list of token strings generated using the comma-separated-values convention.
Arguments:s - The string to separate into tokens sep - the string to use as a delimiter. A single comma character , is used as the default.
