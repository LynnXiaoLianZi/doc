model step command

Syntax

model step i

Execute timesteps. The synonym model cycle may also be used. This is the equivalent of calling model solve with the cycles keyword set to i.

i timesteps are executed unless processing is interrupted by pressing the Shift + Esc, which will terminate cycling, exit the current data file (and all levels of nested data files that may have been used to call it), and return input to the command prompt after the current step is completed.
