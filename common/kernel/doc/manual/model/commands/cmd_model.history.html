model history command

Syntax

model history <name s > keyword

Create a history of a model quantity.

The keyword name may be used to give the history a unique identifier for later reference. If name is not specified the history will be given a name automatically based on the internally assigned ID number.

The following keywords apply:

apply-history-x

applied boundary x history

apply-history-y

applied boundary y history

time-total-creep

history of the creep time.

time-total-fluid

The total accumulated fluid time.

time-step-creep

History of the creep timestep.

solve-local-ratio

The value of the local force ratio.

solve-ratio

The value of the ratio currently being used.

time-total-thermal

The total accumulated thermal time.

time-total

The total accumulated mechanical time.

timestep
The active mechanical timestep.
unbalanced-force

The maximum unbalanced force in any object.

unbalanced-volume

The unbalanced volume in fluid flow.

cycles-total
The total number of cycles accumulated while this process is active.
ratio-average

The value of the average force ratio. See the definition of ratios in ref-equilibrium-.

ratio-maximum

The value of the maximum force ratio. See the definition of ratios in ref-equilibrium-.

mechanical <keyword>
cycle

The total number of mechanical cycle.

energy

History of energy terms.
