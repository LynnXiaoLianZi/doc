program log-file command

Syntax

program log-file s

Specify the log file by name.

Note

As with all program commands, use of the command word program is optional; program log-file and log-file are both valid.

Logging must be activated with the program log on command [DR: check previous commmand is correct] to activate writing. If logging is active and the program.log-file command is given, then the previous log file (if any) is closed and the new file s is opened.
