program playback command

Syntax

program playback s

Play back a record file.

Note

As with all program commands, use of the command word program is optional; program playback and playback are both valid.

When supplying s, if no extension is specified, then “.record” will be assumed.

An ASCII record file representing a “playback” of input to the model is read, theoretically exactly duplicating the steps taken to create the model state from which the record file was derived.

Record files may be generated in a number of ways.

  • They are generated by the “Bundle Pack” command on the “Tools” menu.
  • They may be copied from the output of a model list record commands command
  • They may be copied or saved State Record pane.
  • They may be stripped from the header of a save file.
