fish callback command

Syntax

fish callback keyword

Add/remove functions to be executed in response to callback events.

If the remove keyword is given, then the first instance of sf is removed.

num when arguments
0 at start of every cycle none
1 new contact is created contact index
2 contact is deleted contact index
3 contact with model = 5 or ‘residual’ breaks contact index.
4 local reinforcement calculation element index; normal disp inc; shear disp inc
5 start of each thermal cycle none

Note

The whilestepping expression inside a FISH function automatically inserts the FISH function into the cycle sequence at i = 0. Muti-threading must be turned off for callback 3 to work.
