struct.shell.young

Syntax

f := struct.shell.young(p)
struct.shell.young(p) := f

Get/set the Young’s modulus of the element.

Modifying the stiffness of an element can result in instability if done during cycling. Dependent values should be forced to update using struct.force.update. See the documentation of struct.force.update for comments on the cost.

Returns:f - the Young’s modulus of the element
Accepts:f - the Young’s modulus of the element
Arguments:p - a pointer to a shell element
