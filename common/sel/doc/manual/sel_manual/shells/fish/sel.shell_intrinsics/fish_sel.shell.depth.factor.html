struct.shell.depth.factor

Syntax

f := struct.shell.depth.factor(p)

Get the depth factor at which stresses have been recovered. The depth equals \(Ft/2\) where \(t\) is the shell element thickness. \(F\) values equal to +1.0 or -1.0 correspond to the outer/inner shell surface. The outer surface is defined by the positive \(z\)-direction of the local shell element system. \(F\) values of 0.0 corresponds to the shell mid-surface. The depth factor can be modified with the structure shell recover stress, structure geogrid recover stress, or the structure liner recover stress commands.

Returns:f - the depth factor at which stresses have been recovered
Arguments:p - a pointer to a shell element
