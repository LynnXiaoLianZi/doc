struct.isgroup

Syntax

b := struct.isgroup(p,s<,slot>)

Check if the group s is associated with the structural element and return true if a match is found in any slot. The check for s will only occur in slot, if provided.

A group index or slot index may be used for the arguments s or slot, respectively. Indices operate more efficiently than strings*.

Returns:

b - true if the group was found for that structural element
Arguments:

p - structural element pointer

s - the group name or index*

slot - the slot name or index*; the check occurs on all slots assigned to the structural element if omitted

Note

*Use the group.find and group.find.slot intrinsics to obtain group and slot indices, respectively, and for further information.

Use of the group logic is described in Groups.
