Commands

Operations on structural element nodes.

Nodes are locations where elements interact, and where forces, masses are lumped for equations of motion. All six element types share the same node list. Unlike FLAC3D gridpoints, nodes track six degrees-of-freedom (three rotational ones in addition to three translational).
