structure link slide-tolerance command

Syntax

structure link tolerance-slide f <range>

Set the tolerance used when updating node-to-zone links in response to large-strain motion. A node is considered to be in a zone if it is within a distance :math`delta` of the zone boundary, where :math`delta` equals f times zone size (taken as the maximum \(x\)-, \(y\)-, or \(z\)-dimension of the zone bounding box). By default, f equals 1 Ã— 10-5.
