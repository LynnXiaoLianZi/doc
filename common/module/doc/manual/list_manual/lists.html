List

The term list may appear as a command word. Consider:

zone list |sp3| geometry list |sp3| ball list
fragment list |sp3| block list |sp3| etc.

The term may also be one of the keywords available in a command. See, for instance:

zone water |sp3| fish debug |sp3| zone interface node
block fluid |sp3| model fluid |sp3| etc.

Whether in a command or as a keyword, the functionality is the same: when used, list returns output either to i the Console pane or to a i Listing pane, depending on the length of the information to be returned.

Users familiar with previous versions of the program will recognize this functionality as the old LIST command. However, in common commands in the new syntax, a standalone list command is not provided. Instead, the list logic appears wherever needed—as either command-word or keyword—in model objects that provide list functionality.
