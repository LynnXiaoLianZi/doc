Group

Groups are labels applied to sets of objects, most often by commands pertaining to those objects. The group logic in the program is most commonly used to make it easy to target specific model objects using the range logic. The commands and FISH functions here are provided for performing operations on groups irrespective of the member objects comprising the group, that is: renaming, listing, and finding groups.

The topic Groups provides an overview of how groups work, syntax and variations for group commands, discussion of groups versus named ranges, and other considerations.
