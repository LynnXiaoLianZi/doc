Commands

Commands that operate on the list of group names and slots.

A global list of all group names (in all slots) and all slots created is maintained. It is possible view (group list) and rename (group rename) those items.

Group assignments are made with the group keyword operating on a specific object type (like zone group, geometry polygon group, extrude edge group, etc.).

Group Commands

group create
group list
group rename
group slot
