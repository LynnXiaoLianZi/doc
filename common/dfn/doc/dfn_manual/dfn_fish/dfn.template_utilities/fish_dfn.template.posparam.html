fracture.template.posparam

Syntax

a = fracture.template.posparam(template,i)
fracture.template.posparam(template,i) = a

Get/set a position parameter. The number of parameters can be assessed with the fracture.template.nposparam intrinsic. See the fracture template create command for further details.

Note

Set access is not available for templates that have been used for generation.

Returns:

a - a position parameter
Accepts:

a - a position parameter
Arguments:

template - DFN template pointer

i - index of the position parameter
