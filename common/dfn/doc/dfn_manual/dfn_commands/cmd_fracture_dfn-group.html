fracture dfn-group command

Syntax

fracture dfn-group s keyword <range>

Primary keywords:

remove    slot

Specify DFN group names. Use of the group logic is described in the Groups topic. By default, s is assigned to the default slot Default. Use the remove keyword to remove s from all slots.

remove

Remove s from all slots.

slot slot

Set group slot slot to s. By default, slot Default is used.
