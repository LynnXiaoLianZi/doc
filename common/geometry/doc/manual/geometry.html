Geometry

The geometry system allows the user to create, import and export geometric data. These data can be used to interact with the model in a variety of ways. The data can be used as a filter with the range logic, and as a visualization aid with plotting. FISH extra variables and group names may be assigned to geometric data. FISH may be used to create and manipulate the data. The geometric data are organized into geometric sets, which are named collections of nodes, edges and polygons.

Geometry objects do not require a domain for their creation, since they are not model components. The geometry import command is used to import geometric data from a file that can be of type .stl, .dxf or the Itasca geoemetry format.
