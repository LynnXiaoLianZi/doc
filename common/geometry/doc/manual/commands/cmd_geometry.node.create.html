geometry node create command

Syntax

geometry node create v <keyword ...>

Primary keywords:

extra    group    set

Create a node at the position v in the current geometry set. If the set keyword is given then the node is created in that set.

extra i a

Optionally assign the extra variable a to index i.

group s1 <slot s2 >

Optionally assign the group name s1 to slot s2.

set s

Create the node in the set s.
