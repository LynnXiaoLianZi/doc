plot print-size command

Syntax

plot print-size ix iy

the size of the bitmap used when exporting plot images to certain formats including Postscript, PDF, and SVG. The value assigned at install is 1650 x 1238. This values is saved and restored with the project, and stored in the registry and program shutdown.
