Create the ball-zone coupling data structures. This coupling allows for overlapping balls and zones to experience a kinematic correction.

Note

  • In order for PFC to cycle in a coupled fashion, FLAC3D must be configured for dynamics and run in large strain mode. This is set with the model large-strain and model configure dynamic commands.
