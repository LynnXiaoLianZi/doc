Keyboard

The table below summarizes the keystroke commands available that relate to working with a listing or while a Listing pane is active. If an ellipsis (…) appears after the command label, it indicates that a dialog will appear when the command is used. These commands are contextual, meaning they will produce a different result if a pane other than a Listing pane is active.


Table 1: Listing Pane Key Commands
Keystroke Command Description/Comment
Ctrl+Alt+S Save [listing name] As… Save the listing in the active pane to a file.
Ctrl + W Close [listing name] Close the listing in the active pane. Information in the listing is lost unless it has been previously saved.
Ctrl + C Copy Copy selected text in a listing to the clipboard.
