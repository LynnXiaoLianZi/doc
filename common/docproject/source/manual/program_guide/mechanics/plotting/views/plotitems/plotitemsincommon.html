Plot Items in Common

The following plot items are available to any Itasca numerical modeling code.

Name Description
Axes Reference axes indicating the current orientation of the plot
Fracture Fractures colored by attributes or properties
Fracture Intersections Fracture intersections colored by attributes or properties
Fracture Rosette Fracture rosette chart
Geometry Geometric data
History Chart A 2D chart of one or more histories that have been taken during model execution
History Locations The locations of histories currently being takenâ€”if the history has an associated position
Labels One or more user-defined labels that have been created using the label command
Particle Trace The path taken by traced particles; may also contour the path according to its velocity magnitude
Scalars User-defined scalar data
Table Chart A 2D chart of one or more tables
Tensors User-defined tensor data
Vectors User-defined vector data
