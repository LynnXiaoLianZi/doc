Factor of Safety Plot Item

Description

Show the last-calculated factor of safety in the plot legend.

Attributes

Legend

Legend

Set item labeling in the plot legend. See the standard legend control description for details.

Beneath Title, the Factor of Safety plot item also provides controls in Legend for Value, and Available.

Both of the latter items may be checked, however, they are mutually exclusive: either a last-calculated value is available, and will be shown in the legend, or it isn’t, in which case the “FOS Not Available” message will be shown.
