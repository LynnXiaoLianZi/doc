Program Layout, Configuration, Tools

picture of options dialog

Available program configuration and settings choices available are to be found in the Options dialog. This dialog is organized into a number of “pages” that group logical sets of choices. Almost every pane in the program provides options for preferences or configuration that are found here.

In addition to the Options dialog, user preference is the governing factor behind program layout, access to examples projects, loading program plugins, startup options, and more. Each of these is described in this section.
