Generate Movie Frames

This dialog can be helpful in generating a series of movie frames showing a progression of plot states during a run of a model.

When you run a model with plots in it, you can create save files or result files at specific points. When the model is complete, open this dialog and select the save files or result files from which you want to generate movie frames. You can choose which plots to generate frames for. When you click the Generate button, save/result files will be loaded one by one and images of the plots will be saved for each file.
