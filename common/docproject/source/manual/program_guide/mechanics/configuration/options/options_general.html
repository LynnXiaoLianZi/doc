General Options

Customer Title
The lines entered here appear on all plots, according to user preference set in the Views section of this dialog.
Job Title
The text entered here can be set to appear on all plots, according to the user preference set in the Views section of this dialog.
Echo Input
When checked, includes entered commands (via the command prompt or via data file) as part of the console output.
Log Output
When checked, turns on logging of output to a text file. The user may specify the name of the log and its location using the … button. The current name of the file is indicated next to the check box (“udec.log” in the image above).
Undo Warning Limit (seconds)
An undo command that will take longer than the number of seconds specified here will call a warning dialog. The dialog indicates the estimated time to complete the undo command and allows the user the chance to cancel the command.
Destroy button on left in title bar
When checked, the Close button ( ) on all panes in the program will appear in the title bar at the upper-left corner of the pane; by default (unchecked), they appear in the right corner.
Use system title colors
When checked, the title bar of the active pane will use the base system colors on the computer. By default, this setting is unchecked, and the program uses the colors shown on the swatches below this line to render the title bar of the active pane. The color selected on these swatches may be changed by the user as well.
Don’t show program update notices
When checked, the Update Alert dialog will not appear at program start-up when there is a program update available.
Deterministic by default
When checked, the program is started in Deterministic mode. See topic for further information.
History Step Interval
The value provided here sets the history-sampling step interval. The default value is 10. This value is applied to all histories; different history intervals cannot be assigned to different histories.
Trace Step interval
The value provided here sets the particle-trace-sampling step interval. The default value is 10. This value is used for all traces; different trace intervals cannot be assigned to different traces.
