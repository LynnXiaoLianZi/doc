UDEC

Welcome to the UDEC 7.0 documentation.

This HTML documentation provides information about commands and FISH functions available in UDEC. For the User’s Guide, Theory, Examples and Verification Problems, please see Help -> Manual.

Users of previous versions of UDEC may find the pages UDEC 6.0 to 7.0 Commands Map and UDEC 6.0 to 7.0 FISH Mapping a good place to start before delving into this version’s commands and FISH.

New users should click on the Help menu and choose Getting Started.

Base UDEC Commands

The main commands used by UDEC are encapsulated on the list below. Note these are a mix of UDEC-specific commands and commands that are also commonly used by PFC and FLAC2D.

Table 1: 3DEC Commands
c block c history
c data c model
c dfn c plot
c domain c program
c fish c project
c flow c table
c geometry c trace
c group  

See the Range Phrase Keyword Reference in the range (keyword) section for reference information on range construction.
