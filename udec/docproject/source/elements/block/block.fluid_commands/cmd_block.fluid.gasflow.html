block fluid gasflow command

Syntax

block fluid gasflow b keyword

Primary keywords:

alpha    bulk-minimum    constant    density-minimum    time    timestep-factor    substep-flow    substep-mechanical

Configure gas flow calculations. If off (which is the default), the bulk modulus of non-wetting fluid is pressure independent and is equal to nwbulk. If on, the bulk modulus of non-wetting fluid is equal to pressure in non-wetting fluid multiplied with the coefficient specified in nwbulk.

Keywords below appear in the following formulations:

gas density = B*(pressure)\(^α\); gas bulk modulus =\(\rm{pressure} \over α\).

alpha f

Set the gas constant alpha, used for calculating gas fluid density and bulk modulus.

bulk-minimum f

Set the minimum bulk modulus for gas. Used for zero pressure.

constant f

Set the gas constant B, used for calculating gas fluid density.

density-minimum f

Set the minimum density for gas. Used for zero pressure.

time f

Initial fluid-flow time is set to f for transient flow analysis.

timestep-factor f

Set the fraction of critical timestep to be used for fluid flow analysis to f (default is fw = 1.0).

substep-flow i

Set i fluid-flow timesteps to be performed during each cycle (default: i = 1).

substep-mechanical i

Set the maximum number of mechanical steps executed between fluid-flow steps when nflow is nonzero (see above) (default: n = 1).
