block fluid two-phase command

Syntax

block fluid two-phase <b > keyword

Primary keywords:

minimum-bulk-ratio    time    timestep-factor    saturation-maximum    saturation-minimum

Configure two phase fluid flow.

Turn on two-phase pressure flow calculation (the default is off). Two-phase flow logic can be used with compressible flow only. (See Section 2 in Special Features for a detailed explanation of twophase flow logic in UDEC.)

minimum-bulk-ratio f

Define the minimum ratio between non-wetting fluid bulk modulus and wetting fluid bulk modulus in the case of two-phase flow, SET twophase on; when non-wetting fluid bulk modulus is pressure dependent, SET gas on.

time f

Initial fluid-flow time is set to f for transient flow analysis.

timestep-factor f

Set the fraction of critical timestep to be used for fluid flow analysis (default is f = 1.0).

saturation-maximum f

If the saturation of a domain is larger than f, the domain is considered to be fully saturated in two-phase simulations.

saturation-minimum f

Set the minimum, irreducible saturation of a domain by the wetting fluid in a two-phase calculation. This limit is used because the capillary pressure curve becomes singular for wetting fluid saturation equal to zero.
