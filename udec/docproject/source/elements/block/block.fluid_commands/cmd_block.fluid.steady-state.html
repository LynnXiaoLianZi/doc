block fluid steady-state command

Syntax

block fluid steady-state <b >

Primary keywords:

tolerance-pressure

Configure steady-state flow mode. This algorithm provides faster convergence than the standard transient flow calculation. The command should be used in analyses in which only the steady-state condition (and not the transient response) is required.

tolerance-pressure f

Set domain pressure variation tolerance (default is 0.01).
