block fluid fast-flow command

Syntax

block fluid fast-flow <b > keyword

Primary keywords:

limit-residual-volume    substep-flow    substep-mechanical    time    timestep    tolerance-volume

Configure fast flow calculations (same as incompressible flow).

limit-residual-volume b

Specify control of residual volume limitation in transient fluid flow. The default is off.

substep-flow i

Set i fluid-flow timesteps to be performed during each cycle (default: i = 1).

substep-mechanical i

maximum number of mechanical relaxation steps performed within a fluid cycle (default i = 500)

time

Initial fluid-flow time is set to f for transient flow analysis.

timestep f <fratio >

Fluid timestep is prescribed (required). If the optional fratio is given, the fluid timestep will be multiplied by this factor every flow step within each model cycle command. At the beginning of every model cycle command, the timestep is restored to its initial value, and the process is reinitiated.

tolerance-volume f

Mechanical stepping within a fluid cycle is stopped when the maximum ratio of unbalanced fluid volume to domain volume is below f (default: v = 0.001).
