block structure support command

Syntax

block structure support keyword <range>

Primary keywords:

create    delete    delete-auto    group    list    property

Create and configure a support members. To create the supports, UDEC searches for the two nearest block edges along the axis of a support member, and places the member between the two intersection points. An error will be detected if fewer than two block edges are found. The points of attachment to the blocks are preserved, whatever subsequent displacement occurs.

create keyword

The support element location, geometry and nodal linkage are defined by the following keywords:

location fx fy

fx, fy is one point on the member. This point must be located in empty space.

angle f

The axis of the support member is oriented at a degrees measured counterclockwise from the x-axis (the default is f = 90â—¦).

group s

Assign the created member to the group named s

material i

Material number i is assigned to the support element. The properties are associated with the property number by using the PROPERTY keyword. (The default is i = 1.)

segment i

Support elements with nonzero width may be divided into i segments. An odd number of segments is always placed. (The default is i = 5.)

type keyword

Specify support type.

standard

Specify standard formulation.

rate-dependent

Specify rate-dependent formulation.

width f

The support member spans a width of f perpendicular to its axis. (The default is f = 0.) For example, if width = 10 m and segments = 5, support members are placed at 2.5 m spacing.

delete <range>

Delete the existing support member in the specified range. The keyword must be the only one given in this case.

delete-auto b

If on, structural supports will be automatically deleed if they fail in tension (default is off).

group s

Assign the support member to the group named s.

list

List information regarding support members.

element

List support element data. The column headings are:

(1) index of support element
(2) index of top block
(3) index of bottom block
(4) material number for element
(5) \(x\)-coordinate of element
(6) \(y\)-coordinate of element
(7) normal force on element
(8) normal displacement of element
group s

List support members assigned to the group named s.

property

List structural support properties.

property material i

Set support element properties for material i.

load-rate-exponent

Set exponent constant, \(Î±\).

maximum-force-addition f

Set maximum support force addition, \(Î”Fmax\) [force]

spacing f

Set element spacing in out-of-plane direction (default = 1.0) [length].

stiffness-constant f

Set support stiffness constant, \(k\).

stiffness-axial f /length] [force]

Set axial stiffness of the support member.

If the member contains sub-elements, stiffness-axial is the stiffness for the group of sub-elements. [force/disp.]

If the specified value for stiffness-axial is negative, then a trick table provides the relation between axial force and axial displacement. The table number is the absolute value of the number specified for stiffness-axial. (Note that table should not be used if the support is subject to unloading.)

yield-compression f

Set compressive yield force.

If the member contains sub-elements, yield-compression is the compressive yield force for the group of sub-elements. [force]

yield-tension f

Set tensile strength, \(tmax\) [force]
