block structure liner command

Syntax

block structure liner keyword <range>

Primary keywords:

apply    create    delete    list    node    property

Create, configure, and modify structural element liners. This command is used to automatically apply liner elements on surfaces of the inside of excavations or slope faces. The elements themselves identical to beams, but the generation of the elements is done differently.

apply pressure keyword

The logic used in solving for forces on structural element lining for tunnels is not capable of determining the fluid pressure difference between the inside and the outside of the lining. This command allows the user to specify this pressure difference so that the liner loads will include any hydrostatic forces. The following keywords are available.

inside f

Apply pressure inside tunnel

outside f

Apply pressure at rock liner interface.

The net pressure (inside - outside) is applied to the structural element. It is assumed that the domain fluid pressure, outside, is being applied to the rock.

reference-location fx fy

Set the reference location for pressure gradient. If gravity is active and the water density defined, a pressure gradient may be applied to inside and outside. The values inside and outside are taken as the pressures at the given reference location, and the hydrostatic gradient is superimposed. If the reference location is not given, no gradient is applied.

create keyword
by-angle

Create a structural element liner by specifying a point near the center of an excavation, a starting angle on the perimeter of the excavation and an angle over which to apply the elements. The angles are measured counter-clockwise from the positive x axis.

angle-begin

Set angle of first node (default is 0◦).

angle-span

Set total angle (default is 360◦)

center fx fy

Specify the center point.

connect

Connect the end nodes of the new structural element segment to the endpoints of existing structural elements. The new endpoints must be located within the block corner-round-length of the existing endpoint in order to connect. The result is a single structure, and there will be a jump in the node ID numbers as the redundant node is deleted.

connect-distance f

Set the distance that is used to attempt to connect ends between two structural elements if the connect keyword is used. The default is 2 × corner-round-length.

group s

Assign new structural element segments to the group named s.

length-minimum

Specify the minimum element length (default is 25% of average element length).

length-maximum

Specify the maximum element length.

material-beam i

Specify the element/interface material number.

material-interface i

Specify the material number to assign the interface between two layers of structural elements. The default is material-beam.

points i

Specify the average number of nodes.

by-end-points

Create a structural element liner by specifying ???.

begin fx fy

Specify the location of first node on liner path.

connect

Connect the end nodes of the new structural element segment to the endpoints of existing structural elements. The new endpoints must be located within the block corner-round-length of the existing endpoint in order to connect. The result is a single structure, and there will be a jump in the node ID numbers as the redundant node is deleted.

connect-distance

Set the distance that is used to attempt to connect ends between two structural elements if the connect keyword is used. The default is 2 × corner-round-length.

end fx fy

Specify the location of last node on liner path counterclockwise from first node.

group s

Assign new structural element segments to the group named s.

length-minimum

Specify the minimum element length (default is 25% of average element length).

length-maximum

Specify the maximum element length.

material-beam i

Specify the element/interface material number.

material-interface i

Specify the material number to assign the interface between two layers of structural elements. The default is material-beam.

delete <range>

Delete element segments within the specified range. All segments are deleted if no range is given.

list

List information on structural elements. The following keywords are available.

element

List information on structural element segments. The following optional keywords are available to print information separately.

displacement

List displacements. The column headings are:

(1) structural element index
(2) \(x\)-coordinate of midpoint
(3) \(y\)-coordinate of midpoint
(4) \(x\)-displacement of element
(5) \(y\)-displacement of element
force

List forces. The column headings are:

(1) structural element ID number
(2) axial force (compression positive)
(3) shear force
(4) moment at one end of element
(5) moment at other end of element
(6) failure status
geometry

List geometric data. The column headings are:

(1) structural element ID number
(2) node ID number at one end of element
(3) node ID number at other end of element
(4) material number
(5) length
(6) angle
(7) thickness
(8) area
(9) moment of inertia
(10) cross-section shape factor
group

List group assignments of structural liners.

interface

List structural element interface data. The column headings are:

(1) index of interface contact
(2) material number
(3) constitutive model number
(4) \(x\)-/\(y\)- coordinates of contact
(5) normal/shear forces
(6) normal/shear displacements
(7) ratio of shear to normal force
(8) failure status
(9) length associated with contact
(10) angle of contact surface relative to \(x\)-axis
(11) structural lumped mass/block indexes
node

List information on structural element nodes. The following optional keywords are available to print information separately.

displacement

List displacements. The column headings are:

(1) node ID number
(2) \(x\)-displacement of node
(3) \(y\)-displacement of node
(4) \(x\)-velocity of node
(5) \(y\)-velocity of node
(6) rotational velocity of node
force

List forces. The column headings are:

(1) node ID number
(2) \(x\)-direction force at node
(3) \(y\)-direction force at node
(4) moment at node
(5) scaled mass at node
(6) scaled moment of inertia at node
state

List node state. The column headings are:

(1) node ID number
(2) node index
(3) \(x\)-coordinate of node
(4) \(y\)-coordinate of node
(5) mass of node
(6) moment of inertia of node
enclosed-area

List the total area enclosed by a structural-element lined tunnel. This area can be used as an indication of tunnel closure. The area must be entirely enclosed. If multiple layers of structural elements are used, each layer will add to the total area. If two layers are used, the area printed will be double the actual area.

property

List liner properties

node i
fix-r

Fix the node in the rotation direction.

fix-x

Fix the node in the \(x\)- direction.

fix-y

Fix the node in the \(y\)- direction.

free-r

Free the node in the rotation direction.

free-x

Free the node in the \(x\)- direction.

free-y

Free the node in the \(y\)- direction.

initial

Assign initial values for the following variables.

displacement-x

Set initial \(x\)-displacement value.

displacement-y

Set initial \(y\)-displacement value.

velocity-rotation

Set initial rotational value.

velocity-x

Set initial \(x\)-velocity value.

velocity-y

Set initial \(y\)-velocity value.

force fx fy ifm

Specify specifies nodal applied force and moment.

material i

Specify material number for interface contact (default is 1).

pin

Fix the node in translational space.

property material i

Assign material properties.

coupling-cohesion f

Set interface cohesion [stress].

coupling-dilation f

Set interface dilation [degrees].

coupling-friction f

Set interface friction angle [degrees].

coupling-stiffness-normal f

Set interface normal stiffness [stress/displacement].

coupling-stiffness-shear f

Set interface shear displacement [stress/displacement].

coupling-tension f

Set interface tensile strength [stress].

crack-depth-ratio f

Set crack depth ratio (default = 0).

cross-sectional-area f

Set element area (optional; calculated from thickness, or thickness and width, if not specified) [length2].

density f

Set mass density of element [mass/volume].

moi f

Set second moment of inertia (optional; calculated from thickness, or area and thickness, if not specified) [length4].

moment-thrust-table i

Set table defining the moment thrust diagram.

poisson f

Set Poisson’s ratio.

shape-factor f

Set element shape (optional; default = 5/6; see Figure 1.32 in Special Features).

shear f

shear strength limit [stress]

spacing f

Set element spacing in out-of-plane direction if it isn’t continuous [length] (optional; default = 1.0).

thermal-expansion f

Set thermal expansion coefficient.

thickness f

Set element thickness [length].

width f

Set element width if not continuous in out-of-plane direction [length] (optional; default = 1.0).

yield-compression f

Set tensile yield strength [stress] (default = 1e10).

yield-compression-cracked f

Set residual cracked beam compressive strength [stress] (default = 0.0).

yield-tension f

Set tensile yield strength [stress] (default = 1e10).

yield-tension-residual f

Set residual tensile yield strength [stress].

youngs f

Set elastic modulus [stress].
