block structure rockbolt command

Syntax

block structure rockbolt keyword <range>

Primary keywords:

change    create    delete    fix-tension    free-tension    group    list    node    property

Create, configure, and modify structural element rockbolts. The main difference between cable elements and rockbolt elements is the ability of rockbolt elements to directly resist shear displacements.

change
material i <range>

Change the property numbers for rockbolts located in the specified range.

material-interface i <range>

Change the interface property numbers for rockbolt nodes located in the specified range.

create

Create a rockbolt element.

begin <fx fy > <i >

Specify the start of the rockbolt at coordinate (fx, fy) or at node id i.

connect

Connect the end nodes of the new rockbolt to the end of an existing rockbolt. The new endpoints must be located within the block corner-round-length of the existing endpoint in order to connect. The result is a single structure, and there will be a jump in the node ID numbers as the redundant node is deleted.

connect-distance

Set the distance that is used to attempt to connect ends between two rockbolts if the connect keyword is used. The default is 2 Ã— corner-round-length.

end <f f > <i >

specify the end of the rockbolt at coordinate (fx, fy) or at node id i.

group s

Assign created elements to the group named s.

material i

Assign material number i to the created elements.

segments i

Set the number of segments to i .

table i

Use the specified table to define the end points of the rockbolts.

delete <range>

Delete rockbolt elements in specified range.

fix-tension :flt:'f' <range>

The fix tension will hold the specified axial force constant during cycling. This is used to set a pre-tension force int he cable.

free-tension <range>

The free tension will release a fixed axial force to allow it to change.

group s

Assign the rockbolt element to the group named s.

list

List information about structural element rockbolts.

element

List information on rockbolt element segments. The following optional keywords are available to print information separately.

displacement

displacements. The column headings are:

(1) rockbolt element index
(2) \(x\)-coordinate of midpoint
(3) \(y\)-coordinate of midpoint
(4) \(x\)-displacement of element
(5) \(y\)-displacement of element
force

List forces. The column headings are:

(1) rockbolt element ID number
(2) axial force (compression positive)
(3) shear force
(4) moment at one end of element
(5) moment at other end of element
(6) axial strain
geometry

List geometric data. The column headings are:

(1) rockbolt element ID number
(2) node ID number at one end of element
(3) node ID number at other end of element
(4) material number
(5) length
(6) angle
(7) area
(8) moment of inertia
group s

List rockbolts assigned to the group named s.

node

List information on rockbolt element nodes. The following optional keywords are available to print information separately.

displacement

List displacements. The column headings are:

(1) node ID number
(3) \(y\)-displacement of node
(4) \(x\)-velocity of node
(5) \(y\)-velocity of node
(2) \(x\)-displacement of node
(6) rotational velocity of node
force

List forces. The column headings are:

(1) node ID number
(2) \(x\)-direction force at node
(3) \(y\)-direction force at node
(4) moment at node
(5) scaled mass at node
(6) scaled moment of inertia at node
state

List node state. The column headings are:

(1) node ID number
(2) node index
(3) \(x\)-coordinate of node
(4) \(y\)-coordinate of node
(5) mass of node
interface

List rockbolt structural element interface data. The column headings are:

(1) index of interface contact
(2) material number
(3) constitutive model number
(4) \(x\)-/\(y\)-coordinates of contact
(5) normal/shear forces
(6) normal/shear displacements
(7) ratio of shear to normal force
(8) failure status
(9) length associated with contact
(10) angle of contact surface relative to \(x\)-axis
(11) structural lumped mass/block indexes
property

list rockbolt properties

node i

Create and configure structural element nodes.

create fx fy i

Create a rockbolt element node with ID i at location (fx fy). If the node is within a distance \(d\) of a block edge, it places the node on the block edge (default: \(d\) = corner-round-length).

fix-r

Fix the node in the rotation direction.

fix-x

Fix the node in the \(x\)- direction.

fix-y

Fix the node in the \(y\)- direction.

free-r

Free the node in the rotation direction.

free-x

Free the node in the \(x\)- direction.

free-y

Free the node in the \(y\)- direction.

initial

Assign initial values for the following variables.

displacement-x f

Set initial \(x\)-displacement value.

displacement-y f

Set initial \(y\)-displacement value.

velocity-rotation f

Set initial rotational value.

velocity-x f

Set initial \(x\)-velocity value.

velocity-y f

Set initial \(y\)-velocity value.

force fx fy ifm

Specify specifies nodal applied force and moment.

material i

Specify material number for element and interface contact (default is 1).

pin

Fix the node in translational space.

property material i

Assign material properties for material i .

area f

Set cross-sectional area (rather than radius).

coupling-cohesion-normal f

Set cohesion strength of the rock coupling spring in the normal direction.

coupling-cohesion-shear f

Set cohesion strength of the rock coupling spring in the shear direction.

coupling-cohesion-table i

Set table number for rock coupling spring shear-cohesion softening table.

coupling-confining-table i

Set table relating confining stress factor to deviatoric stress.

coupling-friction-normal f

Set frictional value of the rock coupling spring in the normal direction.

coupling-friction-shear f

Set frictional value of the rock coupling spring in the shear direction.

coupling-friction-table i

Set table number for rock contact spring normal-friction softening table.

coupling-stiffness-normal f

Set stiffness of the rock coupling spring in the normal direction.

coupling-stiffness-shear f

Set stiffness of the rock coupling spring in the shear direction.

density f

Set density of rockbolt element.

moi f

Set moment of inertia of rockbolt element.

perimeter f

Set length of perimeter of hole in which rockbolt is inserted.

plastic-moment f

Set plastic moment.

radius f

Set radius (rather than area) of rockbolt element.

spacing f

Set out-of-plane spacing of rockbolt elements.

tension-failure-strain f

Set axial rupture strain limit,

thermal-expansion f

Set thermal expansion coefficient.

yield-compression f

Set compressive yield force of rockbolt element.

yield-tension f

Set tensile yield force of rockbolt element.

young f

Set Youngâ€™s modulus of rockbolt element.
