block structure beam command

Syntax

block structure beam keyword

Primary keywords:

change    create    delete    list    node    property

Create and manage structural beam elements. Beam elements can only be placed on the perimeters of blocks; they cannot be placed inside of blocks. The coordinates of the ends of each structural element are specified. The beam elements may be constructed by defining nodes and then connecting them with beam elements. Beams may also be constructed by giving the endpoints, either by coordinate pairs or by specifying a table containing the xy-pair coordinates. The following keywords generate create, delete, and configure beam elements.

change keyword <range>

This command is used to change the material property number or constitutive model assigned to a beam element or beam interface.

material-beam i

Change the material number assigned to the beam elements in the specified range.

material-interface i

Change the material number assigned to the beam interface in the specified range.

model-interface i

Change the constitutive model assigned to the beam elements in the specified range. The models available are the same as listed in block contact change.

create keyword

Creates a structural beam element. The beam element nodes may already exist, or may be created at the given location. The element will connect to preexisting elements. If the node is within the corner-round-length of a block edge, it places the node on the block edge.

by-table i keyword

Create beam element nodes at the locations given by the \(xy\)-pairs in table i . The beam elements will be connected to each other. If any node is within the corner-round-length of a block edge, it places the node on the block edge.

connect

Connect the end nodes of the new structural element segment to the endpoints of existing structural elements if given. The new endpoints must be located within the block corner-round-length of the existing endpoint in order to connect. The result is a single structure, and there will be a jump in the node ID numbers as the redundant node is deleted.

connect-distance f

Set the distance that is used to attempt to connect ends between two structural elements if the connect keyword is used. The default is 2 × corner-round-length.

group s

Assign created beam elements to the group named s.

material-beam i

Set the material number for element and interface contact (default: i = 1).

material-interface i

Set the material number to assign the interface between two layers of structural elements. The default is i = material-beam = 1.

no-layer

Prevents a new beam element from connecting to a pre-existing beam element when they overlap.

single keyword

Create a single structural beam element.

begin <i ><fx fy >

Set the location of the beginning node on the beam. The (existing) node may be specified by id (i ) or created at the specified location (fx, fy). The element will connect to preexisting elements. If the node is within the corner-round-length of a block edge, it places the node on the block edge.

end <i ><f f >

Set the location of the end node on the beam. (See above with regard to element and node connectivity.)

connect

Connect the end nodes of the new structural element segment to the endpoints of existing structural elements if given. The new endpoints must be located within the block corner-round-length of the existing endpoint in order to connect. The result is a single structure, and there will be a jump in the node ID numbers as the redundant node is deleted.

connect-distance f

Set the distance that is used to attempt to connect ends between two structural elements if the connect keyword is used. The default is 2 × corner-round-length.

group s

Assign created beam elements to the group named s.

material-beam i

Set the material number for element and interface contact (default: i = 1).

material-interface i

Set the material number to assign the interface between two layers of structural elements. The default is i = material-beam = 1.

no-layer

Prevents a new beam element from connecting to a pre-existing beam element when they overlap.

delete <range>

Structural element segments within the specified range are deleted. The control of the element segment must fall within the range to delete the segment.

list keyword

List information regarding beam elements.

element keyword

List information on beam elements.

displacement

List displacements. The column headings are:

(1) structural element index
(2) \(x\)-coordinate of midpoint
(3) \(y\)-coordinate of midpoint
(4) \(x\)-displacement of element
(5) \(y\)-displacement of element
force

List forces. The column headings are:

(1) structural element ID number
(2) axial force (compression positive)
(3) shear force
(4) moment at one end of element
(5) moment at other end of element
(6) failure status
geometry

List geometric data. The column headings are:

(1) structural element ID number
(2) node ID number at one end of element
(3) node ID number at other end of element
(4) material number
(5) length
(6) angle
(7) thickness
(8) area
(9) moment of inertia
(10) cross-section shape factor
group <s >

List beam element groups.

interface

List structural element interface data. The column headings are:

(1) index of interface contact
(2) material number
(3) constitutive model number
(4) \(x\)-/\(y\)- coordinates of contact
(5) normal/shear forces
(6) normal/shear displacements
(7) ratio of shear to normal force
(8) failure status
(9) length associated with contact
(10) angle of contact surface relative to \(x\)-axis
(11) structural lumped mass/block indexes
node keyword

List information on structural element nodes.

displacement

List node displacements. The column headings are:

(1) node ID number
(2) \(x\)-displacement of node
(3) \(y\)-displacement of node
(4) \(x\)-velocity of node
(5) \(y\)-velocity of node
(6) rotational velocity of node
force

List node forces. The column headings are:

(1) node ID number
(2) \(x\)-direction force at node
(3) \(y\)-direction force at node
(4) moment at node
(5) scaled mass at node
(6) scaled moment of inertia at node
state

List node state. The column headings are:

(1) node ID number
(2) node index
(3) \(x\)-coordinate of node
(4) \(y\)-coordinate of node
(5) mass of node
(6) moment of inertia of node
property

Lists beam properties

node i keyword

Creates and configure a beam element node with ID i.

create fx fy

Create a node at location (fx, fy).

fix-r

Fix node rotation.

fix-x

Fix the node in the \(x\)-direction.

fix-y

Fix the node in the \(y\)-direction.

free-r

Free node rotation.

free-x

Free the node in the \(x\)-direction.

free-y

Free the node in the \(y\)-direction.

initial keyword

Assign initial values to the node.

displacement-x f

Assign initial \(x\)-displacement value.

displacement-y f

Assign initial \(y\)-displacement value.

velocity-rotation f

Assign initial rotational value.

velocity-x f

Assign initial \(x\)-velocity value.

velocity-y f

Assign initial \(y\)-velocity value.

force fx fy fmoment

Specify nodal applied force and moment.

material i

Assign material number for interface contact (default: materal = 1)

pin

Fix the node in translational space.

property material i keyword ...

Assign property values.

coupling-cohesion f

interface cohesion [stress]

coupling-dilation f

interface dilation [degrees]

coupling-friction f

interface friction [degrees]

coupling-stiffness-normal f

interface normal stiffness [stress/length]

coupling-stiffness-shear f

interface shear stiffness [stress/length]

coupling-tension f

interface tensile strength [stress]

crack-depth-ratio f

Set crack depth ratio (default = 0).

cross-sectional-area f

Set element area (optional; calculated from thickness, or thickness and width, if not specified) [length2]

density f

Set mass density of element [mass/volume].

moi f

Set second moment of inertia (optional; calculated from thickness, or area and thickness, if not specified) [length4].

moment-thrust-table i

Set table defining the moment thrust diagram.

poisson f

Set Poisson’s ratio.

shape-factor f

Set element shape (optional; default = 5/6; see Figure 1.32 in Special Features).

shear f

shear strength [stress]

spacing f

Set element spacing in out-of-plane direction if it isn’t continuous [length] (optional; default = 1.0).

thermal-expansion f

Set thermal expansion coefficient.

thickness f

Set element thickness [length].

width f

Set element width if not continuous in out-of-plane direction [length] (optional; default = 1.0).

yield-compression f

Set compressive yield strength [stress] (default = 1e10).

yield-compression-cracked f

Set residual cracked beam compressive strength [stress] (default = 0.0).

yield-tension f

Set tensile yield strength [stress] (default = 1e10).

yield-tension-residual f

Set residual tensile yield strength [stress].

youngs f

Set elastic modulus [stress].
