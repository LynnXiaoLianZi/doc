block structure reinforcement command

Syntax

block structure reinforce keyword <range>

Primary keywords:

create    delete    list    property

Execution of this command creates reinforcing elements across joints that intersect the line that runs from the begin coordinates to the end coordinates. Reinforcement can be used with rigid and deformable blocks. If reinforcement is to be used with deformable blocks, the blocks must be discretized before reinforcement command is used. If the delete keyword is used, all reinforcement elements in the specified range will be deleted.

create keyword ...

Create a reinforcement element.

begin v

Specify the beginning endpoint of a reinforcement element.

end v

Specify the ending endpoint of a reinforcement element.

group s

Assign a created reinforcement element to the group named s.

material i

Specify the material number of a reinforcement element.

id i

Specify the ID number to be used for a reinforcement element.

delete <range>

Remove all reinforcement elements in the specified range.

list <keyword>

List local reinforcing data. The column headings are: not clear this really goes here; do command check

(1) reinforcing contact index
(2) material number for reinforcing
(3) \(x\)-coordinate of reinforcing contact
(4) \(y\)-coordinate of reinforcing contact
(5) axial force in reinforcing
(6) shear force in reinforcing
(7) axial displacement in reinforcing
(8) orientation of reinforcing to joint
(9) 1/2 the active length of reinforcing
(10) index of one block at reinforcement
(11) index of other block at reinforcement
property

List reinforcing element properties.

property mat i

Set properties i for reinforcing elements.

active-half-length f

Set 1/2 “active” length [length].

rupture-tension-strain f

Set axial failure strain (default = 1010).

spacing f

Set spacing of reinforcements in the out-of-plane direction [length] (default = 1.0).

stiffness-axial f

Set axial stiffness [force/length].

stiffness-axial-exponent f

Set axial stiffness exponent (usually 1).

stiffness-shear f

Set shear stiffness [force/length].

stiffness-shear-exponent f

Set shear stiffness exponent (usually 1).

yield-shear-force f

Set ultimate shear capacity (must be > 0) [force].

yield-tension-force f

Set ultimate axial capacity (must be > 0) [force].

The default values for reinforcing properties (except as noted for rupture-tension-strain) are zero.
