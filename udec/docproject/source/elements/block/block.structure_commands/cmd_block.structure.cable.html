block structure cable command

Syntax

block structure cable keyword

Primary keywords:

change    create    delete    fix-tension    free-tension    group    list    material-connect    property    tolerance-move-node

Create, delete, and configure cable elements. The term cable is used loosely and this command may be used to represent any linear reinforcing structure where the shear resistance is provided only by the increased normal stress on the adjoning joints. For shear resistance of the element itself (doweling), the rockbolt element should be used. The cable element typically includes the effects of the axial shear behavior of a grout annulus.

For excavation problems, if the BEGIN coordinates are inside the excavation periphery, the first nodal point will moved to the excavation periphery. The END coordinates should always be located in the rock mass. Cables can only be used with deformable blocks, and the blocks must be zoned prior to defining the cable elements. Pretensioning of the cable elements can be done using the fix-tension and free-tension keywords.

change keyword <range>

This command is used to change the material property number assigned to a cable elements and nodes (grout).

material-grout i

Change grout material property number for all cable nodes in the specified range to i. The material number i must be in the range 1-50.

material-first-node i

Change the grout material number for first node of the cable in the specified range to i. The material number i must be in the range 1-50.

material-last-node i

Change the grout material number for last node of the cable in the specified range to i. The material number i must be in the range 1-50.

material-steel i

Change the steel material property number for all cable elements in the specified range to i. The material number i must be in the range 1-50.

create keyword
begin f f

Set the location of the first node of the cable. If the first node is inside an excavation, it will be moded to the excavation boundary.

end f f

Set the location of the last node on the cable. The last node must lie inside a block zone.

connect

Shift the cable node end closest to a structural (beam or liner) element node to coincide with the beam node. The structural element nodes must be created before the cable nodes. The connection is not allowed to fail. The common, or shared, node will not appear in the list of cable nodes, but it will appear in the list of structural element nodes. The cable element list will show the common node ID number at one end of the cable element, and the normal cable node at the other. Beam and cable nodes are entirely separate entities. This is because the beam nodes are in contact with the surface of a block, whereas the cable nodes are connected to the model in the interior of a zone. All common nodes are therefore structural element nodes, rather than cable nodes. As a consequence, it is possible for the cable element information display to indicate identical node IDs at either end when this keyword is used.

extend

Cause the end of the current cable to connect to the end node of an existing cable element.

group s

Assign created cable elements to the group named s.

nodes

Specify the number of nodes in the cable.

material-grout i

Assign the material property number i for grout.

material-first-node

Assign the material number for the first node of the cable element.

material-last-node

Assign the material number for the last node of the cable element.

material-steel i

Assign the material property number for reinforcing (i.e., steel).

segments i

Specify the number of segements in the cable.

delete <range>

The optional delete keyword will remove all cable elements from the specified range. If no range is given, all cable elements will be deleted.

fix-tension :flt:'f' <range>

The fix tension will hold the specified axial force constant during cycling. This is used to set a pre-tension force int he cable.

free-tension <range>

The free tension will release a fixed axial force to allow it to change.

group s <range>

Assign cable elements within the specified range to the group named s.

list <keyword> <range>

List information regarding cables.

List information regarding cable elements. The column headings are:

(1) cable element index
(2) cable element segment id number
(3) cable node id – first node
(4) cable node id – second node
(5) material number
(6) cross-sectional area
(7) axial force (tension is negative)
(8) length
(9) axial strain (tension is negative)
(10) failure indicator for element:
     0 elastic
     1 at yield in tension
     2 elastic - tension yield in past
     3 at yield in compression
     4 elastic - compression yield in past
     5 failure in extensional strain

List movement information regarding cable nodes. The column headings are:

(1) cable node index
(2) cable node id
(3) \(x\)-coordinate
(4) \(y\)-coordinate
(5) \(x\)-velocity
(6) \(y\)-velocity
(7) \(x\)-displacement
(8) \(y\)-displacement
(9) material number
(10) \(x\)-force applied to node
(11) \(y\)-force applied to node
(12) shear force in grout
(13) node fixed in x if = 1; else 0
(14) node fixed in y if = 1; else 0
(15) failure indicator for node:
     0 elastic
     1 yield
     2 elastic - yield in past
(16) index of zone in which node is currently located

List movement information regarding cable nodes. The column headings are:

(1) cable node index
(2) cable node id
(3) \(x\)-coordinate
(4) \(y\)-coordinate
(5) \(x\)-velocity
(6) \(y\)-velocity
(7) \(x\)-displacement
(8) \(y\)-displacement

List stress and failure information regarding cable nodes. The column headings are:

(1) cable node index
(2) cable node identification number
(3) material number
(4) \(x\)-force applied to node
(5) \(y\)-force applied to node
(6) shear force in grout
(7) node fixed in x if = 1; else 0
(8) node fixed in y if = 1; else 0
(9) failure indicator for node:
     0 elastic
     1 yield
     2 elastic - yield in past
(160) index of zone in which node is currently located
group

Display cable properties.

property

Display cable properties.

material-connect i

Define material property number i for the structural interface node at the location of a connecting cable element.

property material i keyword
cross-sectional-area f

Set cross section area of cable.

density f

Set mass density for cable reinforcing [mass/volume].

grout-stiffness f

Set grout shear stiffness [force/unit cable length/displacement].

grout-strength f

Set grout shear strength [force/unit cable length].

rupture-tension-strain f

Set extensional failure strain (default = 1010).

spacing f

Set spacing of cables in out-of-plane direction (default = 1.0).

thermal-expansion f

Set thermal expansion coefficient for cable.

yield-compression f

Set compressive yield force for cable reinforcing (use positive value) [force].

yield-tension f

Set tensile yield force for cable reinforcing (use positive value) [force].

young f

Set Young’s modulus for cable reinforcing [stress].

Note

The default values for cable properties (except as noted for rupture-tensions-strain and spacing) are zero.

tolerance-move-node f

Having a cable node placed on the edge of a block may cause problems in the calculation of the distributed forces if the node subsequently moves into a different block. To avoid this cable nodes are moved off of the block edges. The distance they are moved defaults to half of the corner-round-length. The tolerance-move-node keyword allows the user to modify the distance the node is moved. The command should be used prior to creating a cable.
