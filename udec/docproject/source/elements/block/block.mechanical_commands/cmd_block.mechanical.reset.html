block mechanical reset command

Syntax

block mechanical reset keyword <range>

Primary keywords:

displacement    joint-displacement    joint-normal-displacement    stress    time    velocity

Calculated model values are set to zero by use of this command. The keywords are:

displacement

All gridpoint and corner displacements in specified range are set to zero.

joint-displacement

All joint normal and shear displacements (in the specified range) are set to zero for plotting or printing incremental changes in displacement. The actual displacements are not reset, because this can affect results if dilation is present.

joint-normal-displacement

All joint normal in the specified range) are set to zero for plotting or printing incremental changes in displacement. The actual displacements are not reset, because this can affect results if dilation is present.

stress

All stresses in blocks and joints in the specified range are set to zero. Zone pore pressures are also set to zero.

time

Current mechanical time is set to zero. This has no effect on the problem being modeled, but is usefulin presenting results. Thermal time is not affected.

velocity

All gridpoint and corner velocities in specified range are set to zero.
