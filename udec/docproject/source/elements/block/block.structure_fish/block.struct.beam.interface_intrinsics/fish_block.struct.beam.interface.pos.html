block.struct.beam.interface.pos

Syntax

Vector Access

v := block.struct.beam.interface.pos(ind<,i>)

Get the position vector for the beam interface. Returns a vector., unless an optional component argument i is supplied, in which case it returns the component value.

Returns:

v - position vector or component
Arguments:

ind - beam interface index

i - optional vector component, from 1 to 2

Component Access

f := block.struct.beam.interface.pos.x(ind)

Get the x-component position.

Returns:f - x-component position.
Accepts:f - x-component position.
Arguments:ind - beam interface index
f := block.struct.beam.interface.pos.y(ind)

Get the y-component position.

Returns:f - y-component position
Accepts:f - y-component position.
Arguments:ind - beam interface index
