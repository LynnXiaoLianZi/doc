block contact initialize command

Syntax

block contact initialize keyword <range>

Primary keywords:

displacement-normal    displacement-shear    force-normal    force-shear    stress-normal    stress-shear

Set initial values for contact variables. The following keywords are available.

displacement-normal f <[initmodifiers]>

Initialize normal displacement for contacts.

displacement-shear f <[initmodifiers]>

Initialize shear displacement for contacts.

force-normal f <[initmodifiers]>

Initialize normal force for contacts.

force-shear f <[initmodifiers]>

Initialize shear force for contacts.

stress-normal f <[initmodifiers]>

Initialize normal stress for contacts.

stress-shear f <[initmodifiers]>

Initialize shear stress for contacts.

Block of Modifiers for the Initial Value

The following keywords are available to modify the value given in the keyword over the range supplied.

add f

Add f (including any specified variations) to existing values of the given variable. The default is to replace the existing value.

mul f

Multiply by f (including any specified variations) the existing values of the given variable by value.

gradient f1 f2

Apply a gradient to the main value, or to the value obtained in conjunction with add or mult. The \(x\) and \(y\) gradient is obtained by the following equation.

\[vtot = value + (gx \cdot x) + (gy \cdot y)\]
