block contact tolerance command

Syntax

block contact tolerance keyword <range>

Primary keywords:

open    overlap    update-location    update-length

Set block contact parameters.

open f

Set the maximum distance a contact may open before it is deleted (default = 0.55 Ã— corner-round-length tolerance).

overlap f

Set the contact overlap tolerance (default: ovtol = one-half corner-round-length tolerance).

update-location i

Update contact locations only every i cycles. (Default is i = 1.)

update-length i

Update contact length only every i cycles (i = 100). The length is also updated every time the model cycle command is issued.
