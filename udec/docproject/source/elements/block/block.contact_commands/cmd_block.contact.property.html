block contact property command

Syntax

block contact property keyword value

Assign properties to material models for contacts. This command is used to define the properties when the block contact cmodel command was used to define the constitutive model. Unless otherwise stated, the default value for all properites is zero.

Table 1: Property keywords for Area Coulomb slip constitutive model
  keyword Description
(1) aperture-residual residual aperture at high normal stress [length]
(2) aperture-zero-load aperture for zero normal stress [length]
(3) cohesion joint cohesion [stress]
(4) constant-b emperical multiplier for fluid flow cubic law (default = 1)
(5) dilation-limit shear displacement at which joint stops dilating. (default=0) (no limit if = 0) [length]
(6) exponent exponent for fluid flow cubic law (default = 3)
(7) friction joint friction [degrees]
(8) friction-residual residual joint friction [degrees]. The residual joint friction (if specified)cannot be exactly equal to zero (use a small nonzero value)
(8) nonwetting-permeabilty-factor non-wetting fluid joint permeability constant (see eq. 2.3 in Special Features Manual). also called joint permeability factor [1/(stress*time)]
(9) permeability-factor wetting fluid joint permeability constant (see eq. 2.3 in Special Features Manual). Also called joint permeability factor [1/(stress*time)]
(10) stiffness-normal joint normal stiffness [stress/length]
(11) stiffness-normal-table joint normal stiffness table
(12) stiffness-shear joint shear stiffness [stress/length]
(13) tension joint tensile strength [stress]

Note

The default value for these properties (with the exception, as noted, of constant-b, exponent and friction-residual) is zero. Joint model 5 is similar to joint model 2, except that when the joint tensile or shear strength is exceeded, then joint tensile strength, joint friction angle and joint cohesion are set to residual values. For joint model 2, joint tensile strength, joint friction angle and joint cohesion are preserved regardless of loading path. To limit fluid flow to broken joint segments use the command block contact crackflow on.

Table 2: Property keywords for Barton Bandis constitutive model
  keyword Description
(1) aperture initial aperture for Barton Bandis joint. This value will be calculated from an emperical formula if not specified [length]
(2) bbdmax maximum value of dilation angle [degrees]
(3) bbdmin minimum value of dilation angle [degrees]
(4) bbdplateau determines whether the damage factors are applied to the plateau region of the shear stress values during shear reversal. Setting this parameter equal to 1 will result in more realistic dilation calculations. The default (bbdplateau = 0) is consistent with the original definition of the Barton-Bandis model.
(5) bbdtable Barton-Bandis shear damage factor table. Up to 10 damage factors may be specified in a table by using the TABLE command. The factors are used to reduce the peak shear for each shear cycle. The first value is normally 1.0 and is applied to shear cycle 1. Each additional value will apply to the next shear cycle. The last value given will be used for all subsequent cycles. The default factors are: 1.0; 0.75; 0.625; 0.5625; and 0.53125
(6) jcso laboratory scale joint wall compressive strength [stress]
(7) jkn joint normal stiffness [stress/length]
(8) jks joint shear stiffness [stress/length]
(9) jrco laboratory scale joint roughness coefficient
(10) ln field scale join length [length]
(11) lo laboratory scale joint length [length]
(12) phir residual friction angle [degrees]
(13) sigmac intact rock uniaxial compressive strength [stress]

Note

The default value for these properties (with the exceptions, as noted) is zero.

Table 3: Property keywords for Continuously Yielding constitutive model
  keyword Description
(1) aperture-residual residual aperture at high normal stress [length]
(2) aperture- zero-load aperture for zero normal stress [length]
(3) constant-b emperical multiplier for fluid flow cubic law (default = 1)
(4) exponent exponent for fluid flow cubic law (default = 3)
(5) exponent-normal exponent of joint elastic normal stiffness
(6) exponent-shear exponent of joint elastic shear stiffness
(7) friction joint friction [degrees]
(8) friction-initial initial joint friction [degrees]
(9) nonwetting-permeabilty-factor non-wetting fluid joint permeability constant (see eq. 2.3 in Special Features Manual). also called joint permeability factor [1/(stress*time)]
(10) permeability-factor wetting fluid joint permeability constant (see eq. 2.3 in Special Features Manual). also called joint permeability factor [1/(stress*time)]
(11) roughness joint roughness parameter [length]
(12) stiffness-normal joint normal stiffness [stress/length]
(13) stiffness-normal-maximum upper limit on joint normal stiffness [stress/length]
(14) stiffness-normal-minimum lower limit on joint normal stiffness [stress/length]
(15) stiffness-shear joint shear stiffness [stress/length]
(16) stiffness-shear-maximum upper limit on joint shear stiffness [stress/length]
(17) stiffness-shear-minimum lower limit on joint shear stiffness [stress/length]

Note

The default values for exponent-normal, exponent-shear, friction-initial and friction-residual are zero. If neither stiffness-normal-minimum nor stiffness-normal-maximum is defined, joint normal stiffness will be constant (i.e., exponent-normal will have no effect). Joint shear stiffness is defined in a similar manner. Timestep calculations for cy joints are based on the values of stiffness-normal-maximum and stiffness-shear-maximum. In general, it will not be necessary to further reduce the timestep for numerical stability. The user should avoid very large values for stiffness-normal-maximum or stiffness-shear-maximum, as these might result in very small timesteps.
