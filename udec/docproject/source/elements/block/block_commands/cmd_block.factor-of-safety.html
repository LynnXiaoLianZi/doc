block factor-of-safety command

Syntax

block factor-of-safety keyword

Primary keywords:

associated    characteristic-steps    filename    interface    list    no_restore    perturbation    ratio    ratio-average    ratio-local    ratio-maximum    resolution    step-limit    structure    zone

UDEC will determine the model mininum factor of safety. This is done through a series of analyses where a strength reduction factor is applied and the model solved to equilibrium. (See section 3.10.5 in the Users’ Guide for more details).

associated

Set solution for block plasticity (default: nonassociated).

characteristic-steps i

Set the characteristic response steps, i. If not specified this is determined using the stress perturbation method.

filename :str:'s'

Set the file name for mode of failure state (default: “FOSMODE.FSV”).

interface keyword
include
exclude

include and exclude keywords each take any of the following keywords.

cohesion

Joint cohesion (default: include).

friction

Joint friction (default: include).

material i

Joints of material i (default: include all).

tension

Joint tension (default: exclude).

list

Displays the factor of safety for a completed analysis.

no_restore b

Controls whether UDEC restores the original model state at the end of the Factor of Safety bracketing. Normally UDEC restores the original model state at the end of the bracketing. If set on the final unstable state will be the current model state. The default is off

perturbation f

Set the stress perturbation factor used in determining the characteristic response steps. (Default is 2.0.)

ratio f

Set the unbalanced force ratio used to determine equilibrium. If this is not explicitly set, UDEC uses the current value used by the previous model solve command (default is 10-5).

ratio-average

Set the ratio limit for mechanical calculations to be defined as the average unbalanced mechanical force magnitude divided by the average applied mechanical force magnitude for all block centroids or gridpoints in the model (default).

ratio-local

Set the ratio limit for mechanical calculations to be defined as the maximum value of the unbalanced mechanical force magnitude divided by the applied mechanical force magnitude for all block centroids or gridpoints in the model.

ratio-maximum

Set the ratio limit for mechanical calculations to be defined as the maximum unbalanced mechanical force magnitude for all block centroids or gridpoints in the model divided by the average applied mechanical force magnitude for all the block centroids or gridpoints.

resolution

Set the resolution in comparing two bracketing (stable and unstable) factors. (The default is .005 × mean value.)

step-limit i

Set a limit to maximum number of cycles (default = 50,000) for FOS calculation.

structure keyword

Include/exclude structural elements in the factor-of-safety calculations.

include
exclude
zone

Include/exclude zone model components in the factor-of-safety calculations.

include
exclude

include and exclude keywords each take any of the following keywords.

cohesion

block cohesion (default: include).

friction

block friction (default: include).

material i

blocks of material i (default: include all).

tension

Block tension (default: exclude).

ucohesion

Ubiquitous joint cohesion (default: include)

ufriction

Ubiquitous joint friction (default: include)
