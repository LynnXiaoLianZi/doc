block apply command

Syntax

block apply keyword <add> <mul> f1 <gradient f2 f3 > <range>

Primary keywords:

force-x    force-y    rotation    velocity-x    velocity-y

Set block or gridpoint variables to initial values. Loads and velocities apply to rigid block centroids only. Initialized values for rigid blocks can be made permanent by using the FIX command. The following keywords are available.

force-x f <[applymodifiers]>

Set rigid block \(x\)-load.

force-y f <[applymodifiers]>

Set rigid block \(y\)-load.

rotation f <[applymodifiers]>

Set both the \(x\)- and \(y\)-component of rigid block \(Î¸\) velocity to v.

velocity-x f <[applymodifiers]>

Set the gridpoint \(x\)-velocity.

velocity-y f <[applymodifiers]>

Set the gridpoint \(y\)-velocity.

Block of Modifiers for the Command Value

The following keyword are available to modify the value given in the keyword over the range supplied. The following keywords are available:

add f

Add f (including any specified variations) to existing values of the given variable. The default is to replace the existing value.

mul f

Multiply by f (including any specified variations) the existing values of the given variable by value.

gradient f1 f2

Apply a gradient to the main value, or to the value obtained in conjunction with add or mult. The \(x\) and \(y\) gradient is obtained by the following equation.

\[vtot = value + (gx \cdot x) + (gy \cdot y)\]
