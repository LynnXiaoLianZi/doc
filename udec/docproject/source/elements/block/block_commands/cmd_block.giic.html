block giic command

Syntax

block giic keyword

Primary keywords:

executable-folder    java-runtime-folder    memory

Invoke the graphical user interface (GIIC) or set GIIC paramters. The use of this command without keywords is only functional using the console version of UDEC. Setting of the GIIC parameters may be done using either version.

executable-folder s

Specifies the path to the GUI folder. (This is usually unnecessary.)

java-runtime-folder s

Specify the path to the Java runtime library folder. (This is usually unnecessary.)

memory i

Specify amount of memory (in MB) to be set aside for the graphical interface. This value is stored in the registry, so the command is only needed to change the amount. (Default = 200 MB.)
