block change command

Syntax

block change keyword <range>

Primary keywords:

material    model

Set or change block material characteristics. All blocks with centroids lying within the optional range have block material characteristics changed. If no range is specified, all blocks will have characteristics changed according to the keywords given below. To change joint characteristics, see block contact change; for cable characteristics, see block struct cable change; for domain characteristics, see block domain change. See the block zone cmodel, block contact cmodel, commands for extended and user-defined zone models, user-defined joint models, and local storage joint models.

The following keywords are used to change block characteristics.

material i

Material property number i is assigned to designated rigid or deformable blocks. (All blocks initially default to mat = 1. The maximum value for i is 50.)

model i

Constitutive number i is assigned to designated deformable blocks according to the following table.

model Model Description
0 null material (The null model is used to model excavated material. The stresses within the null block are automatically set to zero.)
1 linearly elastic, isotropic (default)
3 elastic/plastic, Mohr-Coulomb failure
6 elastic/plastic, Drucker-Prager failure
