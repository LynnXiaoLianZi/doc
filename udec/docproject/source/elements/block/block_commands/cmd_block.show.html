block show command

Syntax

block show keyword <range>

Show hidden blocks. All hidden blocks with centroids within the range specified by the range phrase are made visible.

Most commands only operate on visible blocks.

See range for details on specifying a range. If no range is specified, all hidden blocks are made visible.
