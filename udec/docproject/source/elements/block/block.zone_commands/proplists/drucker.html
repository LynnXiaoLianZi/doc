drucker-prager model properties

drucker-prager
bulk f

bulk modulus, \(K\)

cohesion-drucker f

material parameter, \(k_{\phi}\)

density

mass density, \(ρ\)

dilation-drucker f

material parameter, \(q_{\psi}\). The default is 0.0.

friction-drucker f

material parameter, \(q_{\phi}\)

poisson f

Poisson’s ratio, \(\nu\)

shear f

shear modulus, \(G\)

tension f

tension limit, \({\sigma}^t\). The default is 0.0.

young f

Young’s modulus, \(E\)

Notes:
  • Only one of the two options is required to define the elasticity: bulk modulus \(K\) and shear modulus \(G\), or Young’s modulus \(E\) and Poisson’s ratio \(\nu\). When choosing the latter, Young’s modulus \(E\) must be assigned in advance of Poisson’s ratio \(\nu\).
  • The tension cut-off is \({\sigma}^t = min({\sigma}^t, k_{\phi} / q_{\psi})\).

Plasticity state indicator flags are given below. Use logical and to find individual and multiple state modes.

Bit # State
1 failure in shear now
2 failure in tension now
4 failure in shear in the past
8 failure in tension in the past

See Section 1 in Constitutive Models for details.

Note that the default tension limit is zero for a material with \(q_ϕ\) = 0, and is \(k_ϕ\)/\(q_ϕ\) otherwise. The value assigned for the tension limit remains constant when tensile failure occurs.
