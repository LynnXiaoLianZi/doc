block zone list command

Syntax

block zone list <i > keyword <range>

Primary keywords:

group    mixed    model    pore-pressure    position    property    state    stress    stress-principal

List zone data for deformable blocks. If the zone index i is given, all data for that zone are printed. The following keywords are used to print the information separately. The keyword must be specified.

group

List data associated with zone groups.

(1) zone index
(2) \(x\)-centroid of zone
(3) \(y\)-centroid of zone
(4) group name
mixed

List data associated with mixed discretization.

(1) zone index
(2) mixed discretization code
(3) matched zone
(4) 1st zone in overlay set of zones
(5) 2nd zone in overlay set of zones
model

List data associated with block zone constitutive models.

(1) zone index
(2) \(x\)-centroid
(3) \(y\)-centroid
(4) model name
pore-pressure

List data associated with zone pore pressures and plasticity indicators.

(1) zone index
(2) zone pore pressure
(3) plasticity indicator
    0   elastic
    1   at failure
    2   failed in the past
    4   tension failure
(4) x-coordinate of zone centroid
(5) y-coordinate of zone centroid
position

same as state

property s

List values of specified zone property.

(1) zone index
(2) x-coordinate of zone centroid
(3) y-coordinate of zone centroid
(4) zone model name
(5) property name
(6) property value
state

List block zone state data.

(1) zone index
(2) indexes of gridpoints associated with the zone
(3) zone mass
(4) plasticity indicator
    0    elastic
    1    at failure
    2    failed in the past
    4    tensile failure
stress

List block zone stresses.

(1) zone index
(2) \(xx\)-stress
(3) \(xy\)-stress
(4) \(yy\)-stress
(5) \(zz\)-stress
(6) plasticity indicator
    0    elastic
    1    at failure
    2    failed in the past
    4    tension failure
(7) \(x\)-coordinate of zone centroid
(8) \(y\)-coordinate of zone centroid
stress-principal

List block zone principal stresses.

(1) zone index
(2) major principal stress (\(\sigma\)1)
(3) intermediate principal stress (\(\sigma\)2)
(4) minor principal stress (\(\sigma\)3)
(5) angle of \(\sigma\)1 relative to \(x\)-axis
(6) plasticity indicator
    0    elastic
    1    at failure
    2    failed in the past
    4    tension failure
(7) \(x\)-coordinate of zone centroid
(8) \(y\)-coordinate of zone centroid
